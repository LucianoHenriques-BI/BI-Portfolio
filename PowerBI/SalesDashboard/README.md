## Visão Executiva de Vendas

**Objetivo:**  
Fornecer uma visão rápida da performance de vendas para tomada de decisão executiva.

**KPIs Principais:**
- Receita Total
- Crescimento %
- Ticket Médio
- Volume de Vendas

**Público-alvo:**  
Diretoria e gestão comercial.

**Ferramentas:**
- Power BI
- Modelagem dimensional
- DAX
